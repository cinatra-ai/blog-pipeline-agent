# Blog Pipeline Agent

End-to-end blog generation in a single orchestrated run. Give the agent a short brief and it generates a batch of ideas, drafts the winning one into a polished post, writes matching image prompts, and produces a LinkedIn promo to go with the launch — with operator review along the way so nothing moves forward before you approve it.

## Capabilities

- Generate a batch of distinct blog ideas from a short brief
- Draft the selected idea into a polished markdown blog post
- Write image prompts that match the finished post
- Produce a LinkedIn promo tuned to the new article
- Approve each stage before it advances to the next
